# Impressum

## Herausgeber

**Cloister Protocol**

Cloister Protocol ist ein unabhängiges, auf Compliance ausgerichtetes Privatsphäre-Protokoll. OpenCryptoPay ist seine erste Integration, doch das Protokoll
ist unabhängig vom Zahlungsweg und wird als eigenständige Privatsphäre-Schicht für Stablecoin-Zahlungen auf EVM-
Chains angeboten.
