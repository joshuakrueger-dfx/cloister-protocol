## Removed (GPL-3.0 — eliminated 2026-06-14)

The entire circom/snarkjs toolchain — the only copyleft liability — was removed when the
ZK layer was rebuilt on gnark:

- **`packages/circuits`** (circom sources + `circomlib`, GPL-3.0) — deleted.
- **`packages/prover-webview`** (shipped `snarkjs.min.js`, GPL-3.0, + WebView engine) — deleted.
- **`circomlibjs`, `snarkjs`, `ffjavascript`** npm dependencies — removed from every
  `package.json` (`sdk`, `contracts`, `apps/demo`).
- **`packages/sdk/src/curve.js`** (BabyJubJub via `circomlibjs`) — deleted; the scheme is
  now curve-free (`pubKey = Poseidon2(privKey)`).
- **`poseidon-lite`** (MIT, but circom-ecosystem) — removed; hashing is gnark Poseidon2.

Verification: `grep -rn "snarkjs|circomlibjs|ffjavascript|circomlib" --include=package.json`
returns nothing across the repo, and no product source imports them.

## Trusted setup / cryptographic provenance

The Groth16 proving/verifying keys are generated by `gnark`'s `groth16.Setup` (own run).
The Poseidon2 parameters and BN254 curve come from `gnark-crypto` (Apache-2.0). No
parameters or constants are copied from a GPL source.
